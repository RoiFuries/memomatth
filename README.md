# Memo Matthews - Official Personal Website and Portfolio

> Official repository for Memo Matthews' digital portfolio. Developed entirely with HTML5, CSS3, and vanilla JavaScript for a lightweight, fast, and highly efficient web performance.

## ✨ Features
* **Zero Dependencies:** Built entirely with standard web technologies for maximum speed and simplicity.
* **Custom Error Pages:** Includes personalized `404 Not Found` and `401 Unauthorized` pages to improve user experience.
* **Dynamic Project Templates:** Uses structured HTML templates (`detail_project.html` and `detail_post.html`) to showcase individual portfolio pieces.
* **Responsive Design:** Fully adaptable to different screen sizes.

## 🛠 Technologies Used
The project is built with a focus on core web fundamentals:
* **HTML5:** 86.3% of the codebase
* **CSS3:** 13.7% of the codebase
* **Vanilla JavaScript:** For DOM manipulation and interactivity.
